# Gerador de Quadros de Eventos (v6) — Padrão fixo da planilha

Mapeamento fixo:
- **A** = Evento (ID do bloco)
- **B** = Horário de Concentração
- **D** = Data do evento
- **F** = Previsão para Dispersão
- **H** = Público Estimado (declarado)
- **J** = Percurso
- **K** = Local de Concentração
- **Quantidade de Agentes** = digitar no quadro + escolher cor (vai pro Word)
